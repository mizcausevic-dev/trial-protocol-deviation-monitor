# Security Policy

This repository is a public demonstration surface using **synthetic trial and deviation data only**.

## Scope

- No live PHI, subject data, sponsor data, inspection records, or production deviation packets should be committed here.
- The repo demonstrates deviation-routing and evidence-posture patterns, not a production GxP platform.

## Disclosure

If you identify a security issue in the code or deployment workflow, please open a private report through GitHub Security Advisories if available, or contact the repository owner directly.

## Operational note

This project is **not** represented as GxP-, HIPAA-, 21 CFR Part 11-, or inspection-compliant by default. Any production adaptation would need formal review, validation, data-handling controls, and environment-specific hardening.
